Detailed Description

template<typename T>
struct momentum::VertexNormalDataT< T >

Constraint data for a vertex normal constraint.

Constrains a mesh vertex using a point-to-plane distance where the plane normal can be the source (mesh) normal, the target normal, or a mix of both.
