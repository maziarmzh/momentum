Detailed Description

template<typename T>
class momentum::AimDirErrorFunctionT< T >

The AimDirErrorFunction.

The AimDirErrorFunction computes the element-wise differences between the normalized aim vector and the ray direction. Because the aim vector is normalized, the distance to the ray origin doesn't affect the loss as in AimDistErrorFunction. When the aim vector is close to zero length, we will clamp it to zero.
