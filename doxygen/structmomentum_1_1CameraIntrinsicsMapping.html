Detailed Description

template<typename T>
struct momentum::CameraIntrinsicsMapping< T >

Cached mapping between intrinsic parameters and model parameter indices.

This struct is constructed once (at error function setup time) and caches the index mapping so that solve-time operations avoid string comparisons. It also holds a mutable clone of the intrinsics model that can be updated from model parameters each iteration.
