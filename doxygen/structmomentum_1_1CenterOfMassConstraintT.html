Detailed Description

template<typename T>
struct momentum::CenterOfMassConstraintT< T >

Constraint data for a center of mass target.

Represents a target position for the center of mass (CoM) computed from a weighted sum of joint positions. Each constraint can specify a different set of joints and masses to allow for partial body CoM computation.

Example

CenterOfMassConstraintT<float> constraint;
constraint.jointIndices = {0, 1, 2, 3}; // Spine joints
constraint.masses = {1.0f, 2.0f, 1.5f, 1.0f};
constraint.target = Eigen::Vector3f(0, 1.0f, 0); // Target CoM position
constraint.weight = 1.0f;
