Detailed Description

template<typename S>
struct momentum::CollisionPrimitiveT< S >

Collision geometry primitive.

The tapered capsule fields are kept directly on the primitive so existing code that constructs capsule-only collision geometry can continue to initialize radius, length, parent, and transformation on elements of CollisionGeometry.
