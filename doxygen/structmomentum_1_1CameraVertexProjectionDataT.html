Detailed Description

template<typename T>
struct momentum::CameraVertexProjectionDataT< T >

Per-constraint data for CameraVertexProjectionErrorFunctionT.

Each constraint specifies a mesh vertex (by index) and a target 2D pixel location. The error function penalizes the reprojection error between the vertex's projected position and the target pixel.
