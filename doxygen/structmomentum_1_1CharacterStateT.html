Detailed Description

template<typename T>
struct momentum::CharacterStateT< T >

Represents the complete state of a character at a specific point in time.

Stores the character's parameters, skeleton state, locator positions, skinned mesh, and collision geometry state. This class is used for character animation and rendering.

Updates the character state by applying parameters to referenceCharacter.

Precondition
parameters.pose.size() == referenceCharacter.parameterTransform.numAllModelParameters()
parameters.offsets is either empty or has size referenceCharacter.parameterTransform.numJointParameters(). When empty, it is initialized to zeros.
Parameters
updateMeshIf true, allocate and populate meshState by skinning referenceCharacter.mesh (no-op if mesh or skin weights are null). When pose blend shapes or blend shapes are present on the character, they are applied before linear skinning.
updateCollisionIf true, allocate and populate collisionState from referenceCharacter.collision (no-op if collision is null).
applyLimitsIf true, clamp passive joint parameters to the character's parameterLimits after applying the parameter transform.
