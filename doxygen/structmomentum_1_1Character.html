Detailed Description

A character model with skeletal structure, mesh, and optional components.

Character is intentionally not templated on a scalar type: every member (skeleton, parameter transform, mesh, inverse bind pose, etc.) is stored as float. Double-precision callers convert to/from float at API boundaries via cast<T>() on the inner types (e.g. MeshT<T>, SkeletonStateT<T>). This reflects the reality that the previous CharacterT<T> template parameter was never wired through to any member; see T270138855 for the audit.

Parameters
